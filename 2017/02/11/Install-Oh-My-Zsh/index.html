Install Oh My Zsh

发表于 2017-02-11 | 分类于 iOS | 阅读次数

Install Oh My Zsh

wget --no-check-certificate http://install.ohmyz.sh -O - | sh

View existing shell

cat /etc/shells

Set zsh as default shell

chsh -s /bin/zsh

Configuration

open configuration file vim ~/.zshrc
  • enable self-correction ENABLE_CORRECTION="true"
  • set plugins plugins=(git osx autojump)
  • add this to use autojump
    [[ -s brew --prefix/etc/autojump.sh ]] && . brew --prefix/etc/autojump.sh
open your themes file vim ~/.oh-my-zsh/themes/robbyrussell.zsh-theme

  • set absolute path
