Mac配置Apache服务器

发表于 2017-02-11 | 分类于 iOS | 阅读次数

Apache

  • Mac自带服务器,相对简单快捷
配置步骤：

1.在/Users/用户名目录下新建一个site文件夹
2.打开终端
3.sudo cp /etc/apache2/httpd.conf /etc/apache2/httpd.conf.backup备份httpd.conf文件来避免出错

4.cd /etc/apache2切换路径到apache2
5.sudo vim httpd.conf编辑httpd.conf文件:

  • /DocumentRoot光标定位到DocumentRoot
  • 两处 /Library/WebServer/Documents改为/Users/用户名/site
  • Options FollowSymLinks Multiviews改为Options Indexes FollowSymLinks MultiViews //10.9版本无需改
  • /php光标定位到php后删除行首注释符#
  • :wq保存退出

6.cd /etc目录下输入sudo cp php.ini.default php.ini
7.sudo apachectl -k restart重启apache服务器

  • 配置完在浏览器输入localhost,显示页面目录对应/用户名/site

可能出现问题

  • 页面显示You don’t have permission to access / on this server.
    –httpd.conf文件修改不完全,/DocumentRoot要改两处
  • :wq保存退出时显示readonly
    –编辑httpd.conf时sudo vim httpd.conf的sudo没写
